# Guía de instalación — Guyunusa para Windows

## ¿Por qué Windows muestra una advertencia?

Guyunusa es una aplicación nueva y todavía no cuenta con un **certificado de firma digital** (un trámite costoso que las grandes empresas pagan cada año). Por eso, Windows muestra una advertencia de precaución la primera vez que la instalás.

**Guyunusa es completamente segura.** El aviso no significa que haya un virus — solo que Windows todavía no "conoce" la aplicación porque es reciente. A medida que más personas la instalen, Windows irá reconociéndola automáticamente.

---

## Cómo instalar en 3 pasos

### Paso 1 — Descargá el instalador
Elegí entre:
- **Instalador (.exe)** — se instala como cualquier programa. Recomendado.
- **Versión portable** — no requiere instalación, se ejecuta directamente.

### Paso 2 — Si aparece la pantalla azul de Windows

Vas a ver una ventana que dice **"Windows protegió tu PC"**:

1. Hacé clic en **"Más información"** (texto pequeño, debajo del mensaje)
2. Aparecerá un botón nuevo: **"Ejecutar de todas formas"**
3. Hacé clic ahí y la instalación continúa normalmente

![Ejemplo del aviso](assets/smartscreen-ejemplo.png)

### Paso 3 — Seguí el asistente de instalación
- Elegí la carpeta (o dejá la que viene por defecto)
- Guyunusa se instala y crea un acceso directo en el escritorio
- ¡Listo! Ya podés usarla

## Preguntas frecuentes

**¿Es seguro hacer clic en "Ejecutar de todas formas"?**
Sí. Ese botón existe justamente para aplicaciones legítimas que aún no tienen firma digital. Vos descargaste Guyunusa desde el sitio oficial (guyunusa.uy), así que es seguro.

**¿Por qué la versión portable no da aviso a veces?**
La versión portable no modifica el sistema, así que Windows suele ser menos estricto. Si querés evitar el aviso, probá con la portable.

**¿Necesito permisos de administrador?**
No. Guyunusa se instala solo para tu usuario, sin pedir permisos de administrador.

**¿Cómo desinstalo Guyunusa?**
Desde *Configuración → Aplicaciones → Guyunusa → Desinstalar*, como cualquier otro programa.
