# departamentos-uy — Dominio Departamentos del Uruguay

## Descripción

El dominio **departamentos-uy** es una base documental integral sobre los diecinueve departamentos de la República Oriental del Uruguay. Forma parte del proyecto **BNC-UY** (Base Nacional de Conocimiento Uruguay) y actúa como eje integrador de los demás dominios temáticos.

Cada archivo es completamente autocontenible y está optimizado para recuperación semántica mediante técnicas RAG (Retrieval-Augmented Generation) con búsqueda híbrida BM25 + embeddings semánticos.

## Apartados cubiertos en cada departamento

Cada archivo departamental incluye los siguientes apartados (cuando corresponde):

1. Ficha técnica de identificación
2. Historia y cronología
3. Geografía (relieve, hidrografía, clima, suelos)
4. Naturaleza (flora, fauna, ecosistemas, áreas protegidas)
5. Economía (producción, industria, comercio, turismo)

8. Arquitectura y patrimonio
