# Mamíferos del Uruguay

## Descripción general

Uruguay alberga entre **110 y 120 especies de mamíferos**, que incluyen mamíferos terrestres, semiacuáticos, acuáticos (marinos) y quirópteros (murciélagos). La diversidad mastozoológica refleja la posición de ecotono del Uruguay entre la fauna pampeana templada y los elementos subtropicales del norte.

## Mamíferos terrestres y semiacuáticos — Fichas

### Carpincho (*Hydrochoerus hydrochaeris*)

| Parámetro | Valor |
|-----------|-------|
| **Nombre científico** | *Hydrochoerus hydrochaeris* (Linnaeus, 1766) |
| **Familia** | CAVIIDAE |
| **Orden** | Rodentia |
| **Nombres comunes** | Carpincho, capibara (Brasil), chigüire (Venezuela) |
| **Distribución** | Todo Uruguay, especialmente en riberas de ríos, arroyos y bañados |
| **Hábitat** | Márgenes de cuerpos de agua; humedales |
| **Alimentación** | Herbívoro; gramíneas acuáticas, hierbas, corteza |
| **Tamaño** | 50–65 kg; hasta 130 cm de longitud; el roedor más grande del mundo |
| **Reproducción** | Gestación ~150 días; camadas de 2–8 crías |
| **Estado de conservación** | LC (UICN) |
| **Amenazas** | Caza furtiva para cuero y carne; captura para producción de cuero |
| **Importancia** | Presa de lobito de río, yacaré y rapaces; cicla nutrientes acuáticos |

|-----------|-------|

| **Familia** | MYOCASTORIDAE |
| **Orden** | Rodentia |
| **Nombres comunes** | Coipú, nutria, quiyá |
| **Distribución** | Todo Uruguay; muy común en cuerpos de agua |
| **Hábitat** | Márgenes de ríos, lagunas, bañados |
| **Alimentación** | Herbívoro semiacuático; raíces, tallos, hojas acuáticas |

|-----------|-------|
| **Nombre científico** | *Blastocerus dichotomus* (Illiger, 1815) |
| **Familia** | CERVIDAE |
| **Nombres comunes** | Ciervo de los pantanos, guazú pucú |

### Lobito de río (*Lontra longicaudis*)

| Parámetro | Valor |
|-----------|-------|
| **Nombre científico** | *Lontra longicaudis* (Olfers, 1818) |
| **Familia** | MUSTELIDAE |
| **Nombres comunes** | Lobito de río, nutria de río (nombre confuso; diferente del coipú) |
| **Distribución** | Ríos y arroyos del interior; más frecuente en el norte y este |
| **Hábitat** | Cursos de agua con buena calidad; bordes ribereños con vegetación |

| **Amenazas** | Contaminación de ríos, degradación de riberas, captura accidental en redes |

### Aguará guazú / Lobo de crin (*Chrysocyon brachyurus*)

| Parámetro | Valor |
|-----------|-------|
| **Nombre científico** | *Chrysocyon brachyurus* (Illiger, 1815) |
| **Familia** | CANIDAE |
| **Nombres comunes** | Aguará guazú, lobo de crin, lobo de los esteros |
| **Distribución** | Norte de Uruguay (Artigas, Rivera, Tacuarembó); poco frecuente |
| **Hábitat** | Pastizales altos, humedales, sabanas |
| **Alimentación** | Omnívoro; fruta del lobo (*Solanum lycocarpum*), roedores, aves, reptiles |

| **Importancia** | Dispersor de *Solanum lycocarpum*; indicador de salud del ecosistema |

## Mamíferos marinos

|-----------|-------|

## Quirópteros (murciélagos)

Uruguay tiene aproximadamente **25–30 especies de murciélagos** [VERIFICAR cifra]. La mayoría son insectívoros. Destacan:

- *Tadarida brasiliensis* (murciélago moloso de Brasil) — el más común; colonias en edificios
- *Noctilio leporinus* (murciélago pescador mayor) — atrapa peces en la superficie del agua
- *Eptesicus furinalis*, *Myotis* spp. — insectívoros comunes
