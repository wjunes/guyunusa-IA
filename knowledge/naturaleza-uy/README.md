# naturaleza-uy — Patrimonio Natural del Uruguay

## Descripción

**naturaleza-uy** es un dominio standalone de la Base Nacional de Conocimiento Uruguay (BCN-UY). Documenta el patrimonio natural del Uruguay: biodiversidad, ecosistemas, flora, fauna, geología, áreas protegidas, recursos naturales y conservación ambiental.

El contenido está optimizado para recuperación semántica RAG (Retrieval-Augmented Generation) con búsqueda híbrida BM25 + embeddings. Cada archivo es autocontenido, utiliza nombres científicos, clasificación taxonómica, denominaciones geográficas precisas y marcadores **[VERIFICAR]** para datos que requieren confirmación.

## Alcance temático

- Historia natural de Uruguay y exploradores naturalistas.
- Biodiversidad: flora y fauna nativa, endémica, amenazada e invasora.
- Biomas y ecorregiones del territorio uruguayo.
- Ecosistemas: praderas, humedales, montes, costas, ríos, lagunas.
- Áreas protegidas (SNAP) y sitios Ramsar.
- Geología, geomorfología y paleontología.
- Recursos naturales: agua, suelos, minerales, pesquerías.
- Conservación ambiental, legislación, organismos e instituciones.

## Normas del dominio

- **Formato:** Markdown limpio, sin YAML frontmatter, sin IDs estructurados.
- **Nombres científicos:** siempre en *cursiva*, familia en MAYÚSCULAS cuando corresponda.
- **Estado de conservación:** siglas UICN (LC, NT, VU, EN, CR, EX) con nombre completo.
- **Datos inciertos:** marcados con **[VERIFICAR]**.
- **Fuentes:** Ministerio de Ambiente, SNAP, Facultad de Ciencias (UdelaR), MNHN, DINABISE, UICN, Ramsar, publicaciones científicas revisadas por pares.
- **Sección final:** cada archivo cierra con `## Palabras clave`.
