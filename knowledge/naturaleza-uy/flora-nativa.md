# Flora Nativa del Uruguay

## Descripción general

Uruguay cuenta con aproximadamente **2.500 especies de plantas vasculares nativas**, que representan una notable diversidad en relación con su superficie. La flora nativa se organiza en distintas comunidades vegetales: praderas y pastizales, montes nativos, bosques ribereños, matorrales, palmares, comunidades de quebrada, dunas costeras y vegetación acuática.

La obra de referencia histórica es *Flora Uruguaya* de **José Arechavaleta** (1898–1911); las revisiones más recientes provienen de la **Facultad de Ciencias (UdelaR)** y el **Museo Nacional de Historia Natural (MNHN)**.

## Árboles autóctonos destacados

### Ceibo (*Erythrina crista-galli*)

|-----------|-------|
| **Nombre científico** | *Erythrina crista-galli* L. |
| **Familia** | FABACEAE |
| **Nombres comunes** | Ceibo, seibo |

| **Hábitat** | Márgenes de ríos, arroyos y bañados; suelos húmedos |

### Tala (*Celtis ehrenbergiana*)

| Parámetro | Valor |
|-----------|-------|
| **Nombre científico** | *Celtis ehrenbergiana* (Klotzsch) Liebm. |
| **Familia** | CANNABACEAE |
| **Nombres comunes** | Tala, tala gateado |
| **Distribución** | Todo Uruguay; montes costeros y serranos |
| **Hábitat** | Monte bajo, bosques costeros, serranías calcáreas; suelos pedregosos y arenosos |
| **Características** | Árbol o arbusto de hasta 8 m; espinas; frutos drupáceos |
| **Estado de conservación** | LC |
| **Importancia ecológica** | Especie estructurante del monte nativo costero; refugio y alimento de fauna |
| **Usos tradicionales** | Madera dura usada en postes y herramientas; fruto comestible |

### Coronilla (*Scutia buxifolia*)

| Parámetro | Valor |
|-----------|-------|
| **Nombre científico** | *Scutia buxifolia* Reissek |
| **Familia** | RHAMNACEAE |
| **Nombres comunes** | Coronilla |
| **Distribución** | Todo Uruguay |
| **Hábitat** | Montes nativos, quebradas, laderas rocosas |

| **Estado de conservación** | LC |

| **Familia** | FABACEAE |

### Sauce criollo (*Salix humboldtiana*)

| Parámetro | Valor |
|-----------|-------|
| **Nombre científico** | *Salix humboldtiana* Willd. |
| **Familia** | SALICACEAE |
| **Nombres comunes** | Sauce criollo, sauce colorado |
| **Distribución** | Riberas de ríos y arroyos en todo Uruguay |
| **Hábitat** | Márgenes de cursos de agua |
| **Características** | Árbol de 8–15 m; hojas lineares; corteza rojiza a grisácea |
| **Estado de conservación** | LC |
| **Importancia** | Especie riparia fundamental; controla erosión de márgenes |

### Algarrobo (*Prosopis affinis*)

| Parámetro | Valor |
|-----------|-------|
| **Nombre científico** | *Prosopis affinis* Spreng. |
| **Familia** | FABACEAE |
| **Nombres comunes** | Algarrobo, ñandubay |
| **Distribución** | Noroeste y oeste de Uruguay (Paysandú, Salto, Artigas, Río Negro) |
| **Hábitat** | Monte espinoso, suelos arenosos y secos |
| **Características** | Árbol de hasta 8 m; madera muy resistente; vainas con pulpa dulce |
| **Estado de conservación** | NT (Casi Amenazado) en Uruguay |
| **Importancia** | Madera preciada (muebles, pisos); fruto consumido por fauna y ganado |
| **Usos tradicionales** | Artesanía, postes, carbón; fruto para alimentos y bebidas |

### Ombú (*Phytolacca dioica*)

| Parámetro | Valor |
|-----------|-------|
| **Nombre científico** | *Phytolacca dioica* L. |
| **Familia** | PHYTOLACCACEAE |
| **Nombres comunes** | Ombú |
| **Distribución** | Pastizales de Uruguay, Argentina, sur de Brasil |
| **Hábitat** | Pradera abierta; aislado o en pequeños grupos |
| **Características** | Árbol peculiar con tronco suculento (no es madera verdadera); copa enorme; hasta 20 m de diámetro en el tronco |
| **Estado de conservación** | LC |
| **Importancia cultural** | Símbolo del paisaje pampeano; refugio de fauna en la pradera abierta |

## La Palma Butiá (*Butia odorata*)

| Parámetro | Valor |
|-----------|-------|
| **Nombre científico** | *Butia odorata* (Barb.Rodr.) Noblick |
| **Familia** | ARECACEAE |
| **Nombres comunes** | Palma butiá, butiá, yatay |
| **Distribución** | Principalmente en Rocha; con extensiones a Lavalleja, Maldonado y sur de Brasil |
| **Hábitat** | Praderas sobre suelos arenosos a arcillosos del sureste de Uruguay |
| **Características** | Palma de 5–8 m de altura; hojas pinnadas arqueadas; frutos anaranjados comestibles |
| **Estado de conservación** | VU (Vulnerable) |
| **Importancia** | Los palmares de butiá de Rocha son la mayor concentración de esta especie en el mundo; constituyen un ecosistema propio (palmar de butiá) con fauna asociada |
| **Usos tradicionales** | Fruto para jugos, licores, mermeladas; cogollo comestible; fibra de las hojas |
| **Amenazas** | Forestación con exóticas; falta de regeneración natural (el ganado consume los brotes); escasa incorporación al SNAP |

## Estado de conservación de la flora

La flora nativa uruguaya no cuenta con una Lista Roja completa y actualizada con criterios UICN para todas las especies. Sin embargo, se sabe que:

- Varias especies arbóreas están bajo presión por la extracción de madera y la pérdida de hábitat.
- La palma butiá (*Butia odorata*) es la especie más emblemáticamente amenazada.
- Algunas especies de pastizal especializado (pastizal de flechilla, pastizal serrano) están en retroceso por conversión agrícola.
